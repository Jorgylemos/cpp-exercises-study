# gravity-force-study

## English

This repository contains studies and experiments developed in C++, focusing on building small tools and exploring different concepts as part of the learning process.

Artificial Intelligence tools were used only as a support resource for understanding equations and, occasionally, for brief explanations of specific libraries when documentation was unclear.

Most of the development process relied on official documentation, experimentation, and community resources such as Stack Overflow.

The goal of this repository is to reinforce learning through practice, emphasizing problem-solving and the development of practical skills rather than relying on shortcuts.

## Português

Este repositório contém estudos e experimentos desenvolvidos em C++, com foco na criação de pequenas ferramentas e na exploração de diferentes conceitos como parte do processo de aprendizado.

Ferramentas de Inteligência Artificial foram utilizadas apenas como apoio para entendimento de equações e, ocasionalmente, para explicações rápidas sobre o uso de bibliotecas quando a documentação não era clara.

Grande parte do desenvolvimento foi baseada em documentação oficial, experimentação e recursos da comunidade, como o Stack Overflow.

O objetivo deste repositório é reforçar o aprendizado através da prática, priorizando a resolução de problemas e o desenvolvimento de habilidades práticas, em vez de atalhos.
